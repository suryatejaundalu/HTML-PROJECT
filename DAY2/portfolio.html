HTML

HYPER TEXT MARKUP LANGUAGE(HTML)is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as CSS and scripting languages like JavaScript.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS controls layout, colors, fonts, and responsiveness of web pages.

JavaScript

JavaScript (JS) is a programming language used to create dynamic and interactive web content. It enables features like animations, form validation, and interactive elements on websites. It runs on the client side and is supported by all modern browsers.
